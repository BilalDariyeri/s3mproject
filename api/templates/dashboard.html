PAP / CHAP Kimlik Doğrulama Testi

Kullanıcı adı ve şifre ile giriş denemesi yapın. Bcrypt hash doğrulaması ve Redis rate-limiting mekanizmasını test edin.

Nasıl Çalışır?

1. Kullanıcı adı + şifre → FastAPI /auth endpoint'ine gönderilir

2. PostgreSQL users tablosundan bcrypt hash çekilir

3. bcrypt.checkpw() ile şifre doğrulanır

4. Başarısız → Redis fail:{username} sayacı artırılır

5. 3 hata → 30 saniye blok (rate-limiting)

CLI Eşdeğeri
docker exec nac_freeradius radtest testuser testing123 127.0.0.1 0 testing123
MAC Authentication Bypass (MAB) Testi

802.1X desteklemeyen cihazlar (yazıcı, IP telefon vb.) için MAC adresi tabanlı doğrulama testi.

Kayıtlı MAC Adresleri
Yükleniyor...
Nasıl Çalışır?

1. Cihaz ağa bağlanır — şifre gönderemez

2. Switch, cihazın Calling-Station-Id (MAC) adresini RADIUS'a gönderir

3. FreeRADIUS authorize bölümünde: !User-Password && Calling-Station-Id → MAB akışı

4. FastAPI /auth: users.mac_address sorgulanır

5. Kayıtlı → Accept / Kayıtsız → Reject

CLI Eşdeğeri
echo "User-Name=AA:BB:CC:DD:EE:FF,Calling-Station-Id=AA:BB:CC:DD:EE:FF" | docker exec -i nac_freeradius radclient 127.0.0.1 auth testing123
